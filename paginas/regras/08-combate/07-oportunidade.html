Sumário:

ATAQUE DE OPORTUNIDADE

Ataque de oportunidade é quando o jogador dá a oportunidade do inimigo lhe desferir um ataque sem que você possa esquivar. Além disso o dano será somado com 2 após o ataque.

Os ataques de oportunidade somente ocorrem quando o jogador usa a sua vez de ação para dar as costas e fugir do inimigo. Caso esse inimigo esteja focado no jogador será gerado o ataque de oportunidade.

Caso seja um único inimigo para um grupo de jogadores o mestre deverá usar os dados para decidir se o jogador recebera ataque de oportunidade, podendo jogar 1D10 com ND 7.

NOTA.: O mestre pode também usar 1D2 para decidir isso.
