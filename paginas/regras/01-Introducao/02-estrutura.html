ESTRUTURA DO SISTEMA

O Sistema Venturo é construído em três pilares principais, esses são: As Regras, Criação e Universo. Quando falamos em Sistema Venturo, estamos falando desses três pilares juntos. Um depende do outro para a existência do sistema como um todo.

Regras – As Regra é o pilar onde teremos todas as diretrizes de como usar esse sistema, e uma introdução de como funciona esse universo.

Criação – O pilar da criação se responsabiliza em passar todas as diretrizes para criar elementos para o universo. É nele que você encontra informações de como criar raças, classes, organizações, itens e entre outros elementos que podemos usar na sessão.

Universo – E o último pilar é o próprio universo em si. Tudo que você criar usando o pilar da criação estará localizado no Universo. Esse pilar é subdividido em duas partes, essas são: O próprio universo, e o universo modelo.

O universo modelo é um conjunto de diversos moldes onde você preenche os atributos de cada elemento que você encontra no pilar da criação, que posteriormente será passado para o universo. Então para criar uma raça, teremos toda uma estrutura para ser preenchida e posteriormente alocada no universo.
