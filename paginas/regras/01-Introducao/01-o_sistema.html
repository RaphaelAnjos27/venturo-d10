Regras do Sistema

Agora vamos conhecer as regras por tras do sistema Venturo

Sumário:

O SISTEMA VENTURO D10:

O sistema Venturo D10 foi desenvolvido utilizando princípios de muitos outros sistemas, principalmente o sistema +2D6, criado pelo Tio Nitro.

O sistema Venturo D10 utiliza, em suma, os dados de 10 faces. Nosso sistema usa o número de acertos. Isto facilita e dinamiza a narrativa. Mas não se limita a apenas o uso do D10. Ele também utiliza alguns outros tipos de dados, para a realização de testes específicos.

O nome do sistema provém da forma em que os testes são realizados. Venturo significa “que há de vir, que acontecerá; vindouro”. D10 é a base principal de dados utilizados para jogar os testes do sistema. E em alguns momentos temos o uso de D2, D6, D10 e D20 para definir efeitos (como dano).

O Venturo é um sistema diferenciado dos outros. Ao contrário de outros sistemas, onde você baseia sua campanha em um único mundo (planeta). Aqui você pode, ou não, se basear em uma campanha estelar, que engloba todo um império de raças, podendo também envolver divindade, magia e muitas coisas da fantasia. Ou você pode escolher uma única raça em um período do tempo e criar sua campanha baseada nesse período, por exemplo:

Uma campanha cujo a raça está em um período medieval. E toda a campanha pode se passar apenas dentro de um único continente do planeta daquela raça. Ou essa mesma raça no período futurista, onde a campanha pode ser feita dentro de toda a galáxia. O Sistema Venturo vai sempre tentar dar suporte a esses diversos cenários que um grupo possa querer jogar.

UNIVERSO VENTURO:

O Universo Venturo se trata do universo oficial desse sistema de RPG. Ele é manipulado pela equipe autorizada que desenvolve esse sistema, então todo conteúdo encontrado dentro desse, e dos demais livros relacionados ao sistema, se trata desse universo oficial.

O Sistema é livre, então mestre e jogadores podem pegar e adaptar o sistema a sua vontade, criar e destruir elementos que os agradam. Então lembre-se que os elementos do sistema não é algo travado, você pode customizá-lo a vontade para se adequar ao seu gosto.

OBJETIVO DO SISTEMA VENTURO:

O principal objetivo desse sistema é ser uma ferramenta de criação de mundos coletivos. Com ele você será capaz de criar todo um universo para então criar histórias a partir desses universos.

Então como dito acima, você pode tanto criar um universo inteiro, como pode criar uma raça inteira e jogar somente com ela. E para isso você vai usar as diversas regras de criação e características pre estabelecidas para suas criações como, por exemplo, a linha do tempo universal.

Você deve utilizar de todos os recursos que o sistema oferece para melhora trabalhar nas suas criações, por exemplo, habilidades secundárias. Uma raça inteira pode usar do sistema de habilidades secundárias para gerar as magias da raça, e consequentemente ter todo um escopo histórico para isso. Uma raça pode ser criadora apenas de magias cujo a invocação venha por rituais complexos, pois é daquela forma que eles aprenderam.

TUDO É FICÇÃO:

Embora na sua essência, o Sistema Venturo aborde muitos elementos do cosmo no escopo científico. Muito desses elementos não podem ser levados como verdades absolutas. Lembre-se que esse é um sistema de fantasia, ou seja, muitos elementos aqui foram criados para esse escopo fantasioso, como a possibilidade da existência de Divindades.

Cálculos, valores e padrões usados nesse sistema, muitas vezes não condiz com a realidade (e um especialista na área pode afirmar isso). Então procure usar esse sistema como uma ferramenta de jogo e diversão, e não um material de estudos.

GLOSSÁRIOS

Aqui estão escritas as palavras e termos mais utilizados neste sistema. Algumas palavras são usadas em outros sistemas, mas algumas, não são tão conhecidas, por isso está explicado aqui o significado de cada palavra.

  • RPG – Esta sigla significa “Role-Playing Game”. Sua tradução literal é “Jogo de Interpretar”. Este é o nome dado ao gênero do jogo que aqui apresentamos.
  • Narrador – Também conhecido como “mestre” ou “GM (Game Master)”, é um jogador como os outros, porém é quem conta a história que está acontecendo, descreve o cenário e controla os PnJ (personagens não jogáveis).
  • PnJ – Sigla para “Personagens não Jogáveis”. São os personagens secundários na história, controlados pelo narrador. Antagonistas são exemplos de PnJ. Podemos encontrar esse termo sendo associado também a sigla NPC.
  • PP – Sigla para “Personagem Protagonista”. Esses são os personagens criados e interpretados pelos jogadores.
  • Mesa – É o grupo de jogadores que controlarão os personagens que darão vida ao RPG. Seus jogadores são os protagonistas da história narrada pelo mestre.
  • ND – Sigla para “Nível de Dificuldade”. São os valores mínimos que cada dado da parada deve atingir, para assim, ter sucesso na ação.
  • PD – Sigla para “Parada de Dados”. É a soma dos dados que o jogador terá para jogar, por exemplo, a soma do atributo com a perícia.
  • NV – Sigla para “Nível de Vitória”. Representa o valor de quantos acertos você teve na sua parada de dados.
  • Acertos – É a quantidade de dados que são maior ou igual ao ND em sua PD.
  • Teste – Teste representa quais os atributos e perícias para serem jogadas para manipular algo ou realizar suas ações.
  • VdV – Sigla para “Vitalidade do Veículo”. São os pontos de vida de um veículo qualquer.
  • VM – Sigla para “Velocidade máxima”, muito usado em veículos. Mas você pode acabar vendo em outros contextos.
  • VA – Sigla para “Vigor da Armadura”.
  • PcA – Sigla para Pontos “Críticos da Armadura”.
  • PpD – Sigla para “Pontos Perdidos de Durabilidade”.
  • Origem – A origem no escopo desse RPG está ligada a Raça ou organização originária de determinado objeto ou acontecimento.
  • CG ou Cinematografia – Termo usado no RPG para identificar que o mestre vai fazer uma cinemática sem interação dos personagens para introduzir uma cena.
  • Despertar – Momento onde o personagem passa a ser controlado pelo jogador. Tudo o que acontece com o personagem antes disso se chama “pré-despertar”.
  • Background – É a história pré-despertar do personagem. O jogador deve criar um background para o seu personagem, ou o GM pode fazê-lo.
  • Eventos narrativos – É toda a história ocorrida fora da trama principal que está sendo jogada.
  • Acontecimentos Narrativos – São os eventos que ocorreram dentro da sessão que está sendo jogada.
  • Rolar – É uma palavra usada para especificar que o jogador ou mestre devem rolar uma determinada quantia de dados, independente do atributo ou perícia.
  • Atributo – São área da ficha que determinam características de seu personagem. Habilidades, nome, Aquisições, Vitalidade e por ai vai, são todos atributos da ficha de personagem.
  • Slots – São campos das fichas ao qual o jogador pode preencher. Cada slot equivale a um item do atributo em si, por exemplo, cada habilidade preenchida no atributo Habilidades é um item inserido em um Slot.
  • MtG – é uma sigla para “Morte de Todo o Grupo”. Usado quando todo o grupo morre de uma vez só em uma partida.
  • HtK – é uma sigla usada para “Hit Kill”. Ela é usada para quando um determinado golpe drena toda a vitalidade do alvo o matando instantaneamente.
  • WB – Termo usado para se referir a Worldbuilding, que trata da criação de mundos.
