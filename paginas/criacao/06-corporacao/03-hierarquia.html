HIERARQUIA

A hierarquia é a base da organização de sistemas militares, e serve para compor a cadeia de comando a ser seguida por todos os integrantes do mesmo. Você vai desenvolver as hierarquias de sua organização, tomando como base as áreas dessa organização. E com isso estipular os níveis de hierarquia para cada área.

As hierarquias abordadas aqui foram pegas de um sistema já existentes e podem ser modificadas dependendo do que estiver sendo criado. O Criador da organização pode criar suas próprias hierarquias para adaptá-las ao seu sistema militar.

Essas hierarquias serão criadas para cada uma das áreas de uma organização. E poderão ser usadas para determinar funções dentro dessas áreas. As hierarquias são interessantes de se trabalhar, pois elas além de serem definidas pelo nome, elas definem um cargo. Então pedem uma descrição, e também uma imagem que represente o símbolo da hierarquia.

As hierarquias sempre devem ser listadas do mais importante para o menos importante. Cada uma delas terão os seguintes atributos:

  • Nome da Hierarquia – Qual o nome escolhido para esse posto;
  • Ilustração – Forneça uma ilustração do símbolo da hierarquia;
  • Descrição – De uma descrição das funções de quem tem essa hierarquia e como a mesma pode ser conquistada.
